ReadAsync(CancellationToken)

Reads the next whole (top-level) messagepack data structure.

public ValueTask<ReadOnlySequence<byte>?> ReadAsync(CancellationToken cancellationToken)

Parameters

cancellationToken CancellationToken

A cancellation token.

Returns

ValueTask<ReadOnlySequence<byte>?>

A task whose result is the next whole data structure from the stream, or null if the stream ends. The returned sequence is valid until this MessagePackStreamReader is disposed or until this method is called again, whichever comes first.

Remarks

When null is the result of the returned task, any extra bytes read (between the last complete message and the end of the stream) will be available via the RemainingBytes property.
