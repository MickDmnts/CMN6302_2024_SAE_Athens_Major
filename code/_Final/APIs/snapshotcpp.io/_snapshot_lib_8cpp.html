short cacheData ( unsigned int _smri,
int _dataSize,
unsigned char * _data,
int _refsSize,
int * _refSmris )

That's the main data caching function of the library. Creates and caches the DataContainer with the passed parameters and saves it in the ModelCache. Passed _data are converted into unsigned char values.

Parameters
unsignedint _smri: The smri to associate with the data.
int_dataSize: The size of the marshalled data.
unsignedchar* _data: Pointer to the marshalled data array (byte array)
int_refsSize: The int ref smri array size
int*_refSmris: Pointer to the marshalled int array containing the ref SMRIs of this SMRI.
Returns
OperationSuccessful: If the caching and convertion were successful
OperationFailed: If any error occurs.

◆ packData()

short packData ( )

The main serialization method of the library. The whole _ModelCache gets serialized and stored in the set save path along with a dynamically created name controlled from the SAVE_FORMAT variable. The serialization is handled from the msgpack hpp library.

OperationSuccessful: If the serialization was successful
OperationFailed: If any error occurs.

◆ unpackData()

short unpackData ( )

The deserialization function of the library. The serialized data are read directly from the save path and file name, get deserialized and cached inside the _ModelsCache library map. The GlobalSMRI is set to be equal to the size of the deserialized data map.

Returns
CouldNotOpenFile: In case the file could not be opened.
ReadNotSuccessful: In case the file could not be read.
OperationSuccessful: In case the whole process was successful.
OperationFailed: In any other error.
